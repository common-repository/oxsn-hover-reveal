This plugin adds helpful hover reveal shortcodes with quicktags!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/oxsn-hover-reveal` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->OXSN Hover Reveal screen to configure the plugin
